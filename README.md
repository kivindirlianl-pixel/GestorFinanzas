# GestorFinanzas
Aplicación Progressive Web App (Offline-First) para el control total del presupuesto doméstico.  ✨ Features:  Balance dinámico (Ingresos/Gastos).  Lista de compras con cálculo de ahorro/exceso en tiempo real.  Monitor de Gastos Hormiga.  Formateo de moneda de Colombia (es-CO) con puntos en vivo.
